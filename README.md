# Beyond the Torii – VR神社体験

## 概要
学校の課題で制作したVRで簡単な日本旅行体験ができるアプリケーションです。
ユーザーは仮想空間内で手水や抹茶づくりなどを体験し、直感的な操作を通して日本の文化について学ぶことができます。

## 制作背景
本プロジェクトは、4人の(中国・アラブ・日本)のメンバーで構成された多国籍チームで制作しました。
日本文化(神社)をテーマにしつつ、外国人ユーザーでも直感的に楽しめる体験を目標に設定し、文化的な理解に依存しすぎないUX設計を意識しました。
チーム内では、必ずしも神社の伝統的な要素ではないインタラクション案も多く出されましたが、様々な国籍のユーザーにとって馴染みやすく直感的に楽しめる体験こそが重要であると考え、それらのアイデアを積極的に取り入れています。
結果として、日本らしさを感じられる環境と、多様なインタラクションによる楽しさを両立した体験設計を目指しました。

## 主な機能
- 手を使ったインタラクション（掴む・注ぐなど）
- ステップ形式のチュートリアルガイド
- 神社を再現した没入型環境
- オブジェクトベースの操作設計
- シーン遷移
